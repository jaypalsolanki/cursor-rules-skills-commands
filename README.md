# cursor-rules-skills-commands

`cursor-rules-skills-commands` is a practical repository for building and using Cursor Rules, Skills, and Commands in real software delivery.

Project: `JS.Cursor`

## Why this repository

- Standardize AI-assisted engineering practices across backend, cloud, data, architecture, and delivery.
- Reduce prompt drift by capturing reusable rules and workflows.
- Improve consistency in design reviews, implementation quality, and production readiness.

## What is inside

- `JS.Cursor/Rules`: reusable rule templates and examples (copy into `.cursor/rules/` in any project).
- `JS.Cursor/Skills`: packaged skills (`SKILL.md`) for repeatable engineering tasks.
- `JS.Cursor/Commands`: day-to-day command/prompt playbooks.

## Quick start

1. Read `JS.Cursor/README.md` for adoption guidance.
2. Start from `JS.Cursor/Rules/README.md` and adopt core standards.
3. Add high-value skills from `JS.Cursor/Skills/`.
4. Use command templates from `JS.Cursor/Commands/` in delivery workflows.

## Contribution checklist

- Keep documents concise and execution-focused.
- Include concrete examples, not abstract advice.
- Use consistent terminology across API, DDD, eventing, cloud, and DevOps topics.
- Validate every new asset has clear "when to use" guidance.
