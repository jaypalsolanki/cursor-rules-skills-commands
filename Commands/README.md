# Commands

This folder provides reusable prompt/command templates for daily engineering execution in Cursor.

## Template schema

Each command file should include:

- **Use case**
- **Inputs** (placeholders)
- **Prompt body**
- **Expected output format**

## Starter categories

- API scaffolding and endpoint implementation
- PR risk review
- Distributed incident debugging
- Architecture migration planning
- CI/CD release readiness
- Cloud decision records
