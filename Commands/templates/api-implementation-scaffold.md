# API Implementation Scaffold

## Use case
Generate a production-ready .NET API endpoint plan and implementation checklist.

## Inputs
- Service name: `<service_name>`
- Endpoint: `<method> <route>`
- Domain operation: `<operation>`
- Data store: `<sql_or_nosql>`

## Prompt body
Design and implement `<method> <route>` for `<service_name>` in .NET 8+ using clean architecture and DDD patterns. Include validation, error handling, observability, and test strategy.

## Expected output format
- Endpoint contract
- Layered implementation steps
- Risks and mitigations
- Test cases
