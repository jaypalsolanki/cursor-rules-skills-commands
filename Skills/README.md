# Skills

This folder contains reusable Cursor skills for common engineering workflows.

## Skill anatomy

Each skill lives in its own directory and includes:

- `SKILL.md` (required)
- Optional reference files for deeper guidance

## Authoring guidance

- Keep `SKILL.md` concise and operational.
- Include both **WHAT it does** and **WHEN to use it**.
- Use explicit checklists and expected output templates.
- Prefer one workflow per skill.
